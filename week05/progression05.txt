ใช้ Object.entries() สามารถแปลง object เป็น array ของ key–value pairs ได้ง่าย สามารถนำข้อมูลจาก object มาประมวลผลร่วมกับ array methods ต่างๆได้
Array.isArray() เพื่อตรวจสอบชนิดข้อมูล ว่าข้อมูลที่ได้มาจริง ๆ แล้วเป็น array รึเปล่า
ฝึกการสร้าง Modules ซึ่งเป็นการแยกโค้ดออกเป็นไฟล์ย่อยๆ แล้ว import ไปใช้ใน main.js ทำให้โค้ดอ่านง่าย จัดการได้ดี และสามารถนำกลับมาใช้ซ้ำได้
